# Sprints

Individual sprint implementations and logs for GNW development.

Each sprint is a focused development cycle with a clear objective,
deliverables, and success criteria. Sprints are numbered sequentially
and build on previous work.

## Sprint Structure

Each sprint folder contains:
- SPEC.md — Sprint objective, scope, and success criteria
- LOG.md — Daily/step-by-step development log
- RESULTS.md — Outcomes, metrics, and lessons learned
- rtifacts/ — Code, configs, and other deliverables
