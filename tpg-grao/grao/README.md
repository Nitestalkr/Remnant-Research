# GRAO — Gradient-Driven Research Optimization

The optimization engine that uses computed gradients to drive research direction.

## Core Loop

1. **Collect** signals from TPG
2. **Derive** gradients (directional signals)
3. **Analyze** gradient clusters
4. **Propose** research actions
5. **Execute** proposals
6. **Capture** results as new traces
7. **Repeat**

## Gradient Types

- **Positive gradients**: Patterns that correlate with successful outcomes
- **Negative gradients**: Patterns that correlate with failures or inefficiencies
- **Emergent gradients**: Novel patterns not yet classified

## Memory Integration

GRAO gradients feed into:
- MEMORY.md (long-term memory)
- HEARTBEAT.md (active state)
- Stage contents (capability profiles)
